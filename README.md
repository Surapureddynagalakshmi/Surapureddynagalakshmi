### 🤖 AI Document Assistant — RAG System

> **Intelligent PDF Q&A powered by Retrieval-Augmented Generation**

```
┌─────────────┐    ┌──────────────┐    ┌────────────┐    ┌───────────────┐
│  📄 PDF     │───►│  ✂ Chunking  │───►│ 🧠 Embed   │───►│  🗄 VectorDB  │
│  Upload     │    │  + Overlap   │    │  HuggingFace│    │  ChromaDB     │
└─────────────┘    └──────────────┘    └────────────┘    └───────┬───────┘
                                                                  │
┌─────────────┐    ┌──────────────┐    ┌────────────┐           │
│  💬 Answer  │◄───│  🤖 LLM Gen  │◄───│ 🔍 Semantic│◄──────────┘
│  + Sources  │    │  OpenAI/Llama│    │   Search   │
└─────────────┘    └──────────────┘    └────────────┘
```

**Features:**
- Upload any PDF → instant document intelligence
- Semantic similarity search with sentence embeddings
- Intelligent text chunking with context-preserving overlap
- Vector database for lightning-fast similarity retrieval
- LLM-powered answers with source citations
- Clean FastAPI REST endpoints

**Stack:**
`Python` `FastAPI` `LangChain` `ChromaDB` `FAISS` `HuggingFace Embeddings` `OpenAI / Ollama` `RAG Pipeline`
